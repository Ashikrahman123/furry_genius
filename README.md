# furry_genius

"PET SHOP MODEL"

## Project Overview
furry_genius is a web-based pet shop model application that demonstrates a simple, maintainable front-end and minimal server logic for managing pet listings, showcasing products, and simulating basic pet shop workflows.

## Languages (repo composition)
- JavaScript: 67.5%
- CSS: 22.9%
- HTML: 8.6%
- PHP: 1%

## Features
- Browse and search pet listings
- Responsive UI built with HTML, CSS, and JavaScript
- Client-side interactions and form handling
- Simple backend endpoints (PHP) for demo purposes (if included)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP (optional / demo)
- Data: Local JSON or simple PHP endpoints

## Installation & Local Development
1. Clone the repository:
   git clone https://github.com/Ashikrahman123/furry_genius.git
2. Change into the project directory:
   cd furry_genius
3. If the project is purely static (HTML/CSS/JS), open `index.html` in your browser or serve with a static server (e.g., Live Server in VSCode):
   - npm install -g http-server
   - http-server .
4. If the project includes PHP endpoints, run a local PHP server from the project root:
   php -S localhost:8000

## Usage
- Open the app in your browser at http://localhost:8080 (or http://localhost:8000 for PHP server)
- Explore the pet listings, use the search/filter controls, and try adding sample entries if the UI supports it.

## Project Structure (typical)
